# Echoes of the Forgotten

Echoes of the Forgotten é um jogo 3D de exploração e puzzles desenvolvido na Unity como projeto acadêmico.

## Funcionalidades

- Movimentação em terceira pessoa
- Sistema de inventário
- Sistema de diálogos
- Sistema de save por checkpoints
- Coleta de itens persistente
- Puzzles com botões
- Progressão baseada em exploração
- Cutscenes
- Troca de salas

## 🛠 Tecnologias

- Unity
- C#
- TextMesh Pro
- JSON (Save System)

## Como executar

1. Clone o repositório:

```bash
git clone https://github.com/pedrobarbosa239/Echoes-of-the-Forgotten.git
```

2. Abra o projeto na Unity Hub.
3. Abra a cena inicial.
4. Pressione **Play**.
